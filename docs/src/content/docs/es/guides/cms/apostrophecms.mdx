[ApostropheCMS](https://apostrophecms.com/) es un sistema de gestión de contenidos que admite la edición en la página en Astro.

## Recursos de la comunidad

- [Integración de Astro para ApostropheCMS](https://apostrophecms.com/extensions/astro-integration) - Plugin de Astro para ApostropheCMS, guía de integración y proyectos de inicio tanto para Apostrophe como para Astro.
