> **CouldNotTransformImage**: No se puede transformar la imagen `IMAGE_PATH`. Consulta la traza de la pila para obtener más información.

## ¿Qué salió mal?
Astro no pudo transformar una de sus imágenes. A menudo, esto se debe a una imagen dañada o mal formada. Volver a exportar la imagen desde su editor de imágenes puede solucionar este problema.

Dependiendo del servicio de imágenes que esté utilizando, la traza de la pila puede contener más información sobre el error específico encontrado.
