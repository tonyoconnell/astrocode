> **MissingLocale**: La locale/chemin `LOCALE` n'existe pas dans le fichier configuré `i18n.locales`.

## Qu'est-ce qui n'a pas fonctionné ?
Astro ne trouve pas la locale demandée. Toutes les locales supportées doivent être configurées dans [i18n.locales](/fr/reference/configuration-reference/#i18nlocales) et avoir des répertoires correspondants dans `src/pages/`.
