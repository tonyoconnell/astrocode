> **CouldNotTransformImage**: Impossible de transformer l'image `IMAGE_PATH`. Voir la trace de la pile pour plus d'informations.

## Qu'est-ce qui n'a pas fonctionné ?
Astro n'a pas pu transformer l'une de vos images. Souvent, cela est dû à une image corrompue ou malformée. La réexportation de l'image à partir de votre éditeur d'images peut résoudre ce problème.

Selon le service d'image que vous utilisez, la trace de pile peut contenir plus d'informations sur l'erreur spécifique rencontrée.
