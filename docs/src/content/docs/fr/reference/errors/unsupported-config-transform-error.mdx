> **UnsupportedConfigTransformError** : Les fonctions `transform()` dans votre configuration de contenu doivent renvoyer un JSON valide, ou des types de données compatibles avec la bibliothèque devalue (y compris Dates, Maps et Sets).<br/>Erreur complète : PARSE_ERROR 

## Qu'est-ce qui ne va pas ? 

Les fonctions `transform()` dans votre configuration de contenu doivent renvoyer un JSON valide ou des types de données compatibles avec la bibliothèque devalue (y compris Dates, Maps et Sets).

-  [librarie devalue](https://github.com/rich-harris/devalue)
