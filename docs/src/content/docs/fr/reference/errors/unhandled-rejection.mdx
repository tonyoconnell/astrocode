> **UnhandledRejection**: Astro a détecté un rejet non traité. Voici la trace de la pile:<br/>STACK
> 
## Qu'est-ce qui n'a pas marché ?
Astro n'a pas pu trouver de code pour gérer une `Promise` rejetée. Assurez-vous que toutes vos promesses ont un gestionnaire `await` ou `.catch()`.
