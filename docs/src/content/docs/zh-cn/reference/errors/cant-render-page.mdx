> **CantRenderPage**: Astro 无法找到任何内容来渲染此路由。此路由没有关联的文件或重定向。

## 出了什么问题？
Astro 在尝试渲染路由时无法找到关联的文件内容。这是 Astro 的错误，而不是用户的错误。如果重启开发服务器不能解决问题，请提交一个 issue。
