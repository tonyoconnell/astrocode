> **UnhandledRejection**：Astro 检测到未经处理的 rejection。以下是堆栈跟踪：<br/>STACK

## 出了什么问题？
Astro 找不到任何代码来处理被拒绝的 `Promise`。确保你的所有 promises 都有一个 `await` 或 `.catch()` 处理程序。
