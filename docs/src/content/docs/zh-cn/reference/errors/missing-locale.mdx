> **MissingLocale**: 语言环境/路径 `LOCALE` 在配置的 `i18n.locales` 中不存在。

## 出了什么问题？
Astro 找不到请求的语言环境。所有支持的语言环境必须在 [i18n.locales](/zh-cn/reference/configuration-reference/#i18nlocales) 中配置，并在 `src/pages/` 中有对应的目录。
