> **NoPrerenderedRoutesWithDomains**: 当启用多个域名时，尚不支持静态页面的预渲染功能。如果你希望启用此功能，你必须禁用页面 COMPONENT 的预渲染。

## 出了什么问题？
在启用国际化域名的情况下，静态页面尚未得到支持。如果你希望启用此功能，必须关闭预渲染。
