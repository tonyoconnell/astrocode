> **CouldNotTransformImage**：无法转换图像 `IMAGE_PATH`。有关更多信息，请参阅堆栈跟踪。

## 哪里出了问题？

Astro 无法转换你的图像。通常，这是由于图像损坏或格式不正确。重新从图像编辑器导出图像可能会解决此问题。

根据你使用的图像服务，堆栈跟踪可能包含有关遇到的特定错误的更多信息。
