> **오류 메시지 예시:**<br/>
InvalidPrerenderExport: A `prerender` export has been detected, but its value cannot be statically analyzed.

## 무엇이 잘못되었나요?

`prerender` 기능은 유효한 JavaScript의 하위 집합만 지원합니다. `export const prerender = true`를 정확하게 사용하여 컴파일러가 빌드 시 이 지시어를 감지할 수 있도록 만들어야합니다. 변수 `let` 및 `var` 선언은 지원되지 않습니다.
