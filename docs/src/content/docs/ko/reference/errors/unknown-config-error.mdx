## 무엇이 잘못되었나요?

Astro 구성 파일을 로드하는 중 Astro에서 알 수 없는 오류가 발생했습니다.
이는 구성의 구문 오류로 인해 발생하는 경우가 많으며 메시지는 더 많은 정보를 제공해야 합니다.

이 오류가 발생하는 이유를 알고 있다면 [문제를 보고](https://astro.build/issues/)해주시면 감사하겠습니다.
