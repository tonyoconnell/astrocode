> **InvalidGlob**: Invalid glob pattern: `GLOB_PATTERN`. Glob patterns must start with './', '../' or '/'.

## 무엇이 잘못되었나요?

Astro가 잘못된 glob 패턴을 발견했습니다. 이는 glob 패턴이 유효한 파일 경로가 아니기 때문에 발생하는 경우가 많습니다.
