> **CouldNotTransformImage**: Could not transform image `IMAGE_PATH`. See the stack trace for more information.

## 무엇이 잘못되었나요?

Astro가 이미지 중 하나를 변환할 수 없습니다. 종종 이는 손상되거나 잘못된 형식의 이미지로 인해 발생합니다. 이미지 편집기에서 이미지를 다시 내보내면 이 문제가 해결될 수 있습니다.

사용 중인 이미지 서비스에 따라, 발생한 특정 오류에 대한 추가 정보가 스택 추적에 포함될 수 있습니다.
