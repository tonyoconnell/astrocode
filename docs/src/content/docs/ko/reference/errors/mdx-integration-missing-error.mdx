> **MdxIntegrationMissingError**: Unable to render FILE. Ensure that the `@astrojs/mdx` integration is installed.

## 무엇이 잘못되었나요?

공식 `@astrojs/mdx` 통합을 찾을 수 없습니다. 이 오류는 MDX 통합이 설치되지 않은 상태에서 MDX 파일을 사용할 때 발생합니다.
