> **UnsupportedImageConversion**: Converting between vector (such as SVGs) and raster (such as PNGs and JPEGs) images is not currently supported.

## 무엇이 잘못되었나요?

Astro는 현재 벡터 (예: SVG) 이미지와 래스터 (예: PNG 및 JPEG) 이미지 간 변환을 지원하지 않습니다.
