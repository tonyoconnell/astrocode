> A collection queried via `getCollection()` does not exist.

## 무엇이 잘못되었나요?

컬렉션을 쿼리할 때 요청된 이름의 컬렉션 디렉터리가 `src/content/` 디렉터리 아래에 있는지 확인하세요.
