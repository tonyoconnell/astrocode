> **ImageNotFound**: Could not find requested image `IMAGE_PATH`. Does it exist?

## 무엇이 잘못되었나요?

Astro가 가져온 이미지를 찾을 수 없습니다. 종종 이는 단순히 경로의 오타로 인해 발생합니다.

Markdown의 이미지는 현재 파일을 기준으로 합니다. `.md` 파일과 같은 폴더에 있는 이미지를 참조하려면 경로가 `./`로 시작해야 합니다.
