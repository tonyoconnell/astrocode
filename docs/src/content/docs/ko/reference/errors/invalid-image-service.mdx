> **InvalidImageService**: There was an error loading the configured image service. Please see the stack trace for more information.

## 무엇이 잘못되었나요?

구성된 이미지 서비스를 로드하는 중 오류가 발생했습니다. 이는 이미지 서비스가 기본 내보내기에서 호환 객체를 제대로 내보내지 않거나 잘못된 경로 등 다양한 요인으로 인해 발생할 수 있습니다.

서비스가 올바르게 구성되었으며 이 오류가 잘못되었다고 생각되면 [문제를 보고해주세요](https://astro.build/issues/).
