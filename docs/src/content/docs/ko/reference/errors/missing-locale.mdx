> **MissingLocale**: The locale/path `LOCALE` does not exist in the configured `i18n.locales`.

## 무엇이 잘못되었나요?

Astro가 요청한 로케일을 찾을 수 없습니다. 지원되는 모든 로케일은 [i18n.locales](/ko/reference/configuration-reference/#i18nlocales)에서 구성되어야 하며 해당 디렉터리가 `src/pages/` 디렉터리 내부에 존재해야합니다.
