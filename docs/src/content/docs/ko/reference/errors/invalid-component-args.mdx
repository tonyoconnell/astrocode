> **오류 메시지 예시:**<br/>
InvalidComponentArgs: Invalid arguments passed to `<MyAstroComponent>` component.

## 무엇이 잘못되었나요?

Astro 컴포넌트는 `Component()` 또는 `{items.map(Component)}`와 같은 함수 호출을 통해 수동으로 렌더링할 수 없습니다. 컴포넌트 구문 `<Component />` 또는 `{items.map(item => <Component {...item} />)}`을 선호합니다.
