## 무엇이 잘못되었나요?

Astro에서 CLI 명령 중 하나를 시작하는 동안 알 수 없는 오류가 발생했습니다. 오류 메시지에는 더 많은 정보가 포함되어야 합니다.

이 오류가 발생하는 이유를 알고 있다면 [문제를 보고](https://astro.build/issues/)해주시면 감사하겠습니다.
