> **FailedToLoadModuleSSR**: Could not import `IMPORT_NAME`.

## 무엇이 잘못되었나요?

Astro가 요청한 파일을 가져올 수 없습니다. 이는 종종 가져오기 경로가 잘못되어 발생합니다 (파일이 존재하지 않거나 경로에 오타가 있기 때문).

이 메시지는 [타입 가져오기](/ko/guides/typescript/#타입-가져오기)임을 지정하지 않고 타입을 가져온 경우에도 나타날 수 있습니다.
