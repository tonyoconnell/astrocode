> Could not process image metadata for `IMAGE_PATH`.

## 무엇이 잘못되었나요?

Astro가 가져온 이미지의 메타데이터를 처리할 수 없습니다. 이는 손상되거나 잘못된 형식의 이미지로 인해 발생하는 경우가 많으며 이미지 편집기에서 이미지를 다시 내보내면 이 문제가 해결될 수 있습니다.
