Astro 커뮤니티의 구성원은 여러 타사 도구 및 서비스를 Astro 웹 사이트에 성공적으로 통합했습니다!

다음은 Astro를 다양한 인기 CMS, 전자 상거래, 인증/권한 부여, 검색 및 댓글 기술과 연결할 수 있는 방법을 보여주는 커뮤니티의 일부 배포 사이트, 저장소, 블로그 게시물, 동영상입니다.

:::tip[특정 통합을 찾고 계신가요?]
새로운 [통합 라이브러리](https://astro.build/integrations/)를 확인하고 라이브러리에 [통합을 추가](/ko/reference/publish-to-npm/#통합-라이브러리)하는 방법을 알아보세요!
:::

## 저장소 / 시작 템플릿

`create astro` 명령에 `--template` 인수를 전달하여 기존 github 리포지토리를 기반으로 새 astro 프로젝트를 시작할 수 있습니다.

<PackageManagerTabs>
  <Fragment slot="npm">
  ```shell
  npm create astro@latest -- --template <github-username>/<github-repo>
  ```
  </Fragment>
  <Fragment slot="pnpm">
  ```shell
  pnpm create astro@latest --template <github-username>/<github-repo>
  ```
  </Fragment>
  <Fragment slot="yarn">
  ```shell
  yarn create astro --template <github-username>/<github-repo>
  ```
  </Fragment>
</PackageManagerTabs>
